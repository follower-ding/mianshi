# Variant: 暗色科技 Dark Tech

**Mood:** 专注、夜间编码、技术感 — 像 Raycast / Height 的暗色 productivity，非赛博朋克 HUD。

## Colors

| Role | Hex | Variable |
|------|-----|----------|
| Primary | `#E2E8F0` | `--color-primary` |
| Accent | `#22D3EE` | `--color-accent` |
| Background | `#0A0A0F` | `--color-background` |
| Surface | `#12121A` | `--color-surface` |
| Surface Elevated | `#1A1A24` | `--color-surface-elevated` |
| Text | `#F1F5F9` | `--color-text` |
| Muted | `#94A3B8` | `--color-text-muted` |
| Border | `#27272A` | `--color-border` |

## Typography

- Display/Heading: **Inter**
- Body: **Inter**
- Code/Data: **JetBrains Mono**

## Style Rules

-  subtle grid/dot 背景 3% opacity
- Accent cyan 仅用于 focus、active、数据高亮 — 不全页 glow
- Glass 卡片 `bg-white/5` + border `white/10` + blur 24px（暗色专用）
- 等宽字体用于面试对话/代码块

## Anti-AI for this variant

- 禁止 matrix 绿 + 满屏 scanline
- 禁止 purple gradient hero
- 禁止过多 neon 描边

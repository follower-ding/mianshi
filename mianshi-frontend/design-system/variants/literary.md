# Variant: 文艺 Literary

**Mood:** 温暖、编辑感、有温度 — 像翻阅职业成长笔记，非冷峻工具盘。

## Colors

| Role | Hex | Variable |
|------|-----|----------|
| Primary | `#5C534A` | `--color-primary` |
| Accent | `#B8956B` | `--color-accent` |
| Background | `#FAF8F5` | `--color-background` |
| Surface | `#FFFFFF` | `--color-surface` |
| Text | `#2C2825` | `--color-text` |
| Muted | `#6B6158` | `--color-text-muted` |
| Border | `#E8E2DA` | `--color-border` |

## Typography

- Display/Heading: **Lora** (衬线)
- Body: **Raleway**
- Pull quote / 引语: Lora italic

## Style Rules

- 单列阅读宽 max 680px；大留白 section py-24+
- 不对称 editorial grid；细分割线非 heavy shadow
- 动效 200–300ms ease-out；无 neon
- 阶段/章节用 muted 色带区分

## Anti-AI for this variant

- 禁止 cyber/neon/暗色底
- 禁止 sans-only 大标题居中 slogan
- 禁止 emoji 装饰
